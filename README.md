# 📝 Noted – Simple Notes Web App

A lightweight browser-based notes app built with HTML, CSS, and JavaScript.  
It allows you to create, edit, rename, and delete notes, with all data saved locally in your browser. 
Built as a learning project using vanilla HTML, CSS, and JavaScript, with AI-assisted guidance during development.

---

## 🚀 Features

- 🆕 Create new notes
- 💾 Save notes instantly
- ✏️ Edit existing notes
- 🔄 Notes persist after refresh (localStorage)
- 🏷 Rename notes
- 🗑 Delete notes
- 📋 Simple sidebar for navigation

---

## 📂 How It Works

- Notes are stored in your browser using localStorage
- Each note has:
  - Title (first line of the note)
  - Content (full text)
- Sidebar updates automatically when notes are added, renamed, or deleted

---

## 🎮 User Controls

### 🆕 Create a New Note
Click the **New Note** button  
This clears the editor so you can start a fresh note.

---

### 💾 Save a Note
Click the **Save Note** button

- Saves the current text in the editor
- If a note is already open, it will be updated
- If not, a new note is created

---

### 📂 Open a Note
Click any note in the left sidebar  
This loads the note into the editor.

---

### ✏️ Rename a Note
Double click a note in the sidebar

- A prompt will appear
- Enter the new name
- Press OK to update

---

### 🗑 Delete a Note
Right click a note in the sidebar

- A confirmation box appears
- Click OK to delete the note permanently

---

## 📌 Example Workflow

1. Click New Note
2. Type your note
3. Click Save Note
4. See it appear in the sidebar
5. Click it to reopen
6. Double click to rename
7. Right click to delete

---

## ⚠️ Note

All data is stored locally in your browser.  
Clearing browser storage will delete all notes.
